Text Alert System

Introduction: 

        The Text Alert System is a computer program coded into a raspberry pi that enables users to send reminder texts. A remind text is a pre programmed sms messages that automatically sends itself at a user specified date and time. This system was designed for myself for the purpose of aiding in reminding Key Club members when they signed up to volunteer.

Scope:

The final project will be accessed through a website where a user can enter a receiving number, message, month, day, hour, and minute to create a remind text that will be sent automatically.

Deliverables:

  1. Access to twilio api to send text messages
  2. Raspberry Pi, monitor, power cables, hdmi cable, wifi dongle, mouse, keyboard
  3. Python, IDE, xampp/docker for webserver and database
  4. Method for delaying the sending of remind texts
  5. Database that can handle datetime, strings, and ints
  6. Raspberry pi can be accessed by other devices
  7. Website can access database to store information
